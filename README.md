### What I'm working on

- [mojave](https://github.com/carlosrabelo/mojave) -- Modular C++ CPU emulator for retro architectures
- [gobi](https://github.com/carlosrabelo/gobi) -- A dBase II clone written in Go with a VT100 TUI
- [pbasic](https://github.com/carlosrabelo/pbasic) -- Educational BASIC interpreter in MIPS assembly
- [bip38cli](https://github.com/carlosrabelo/bip38cli) -- Bitcoin private key encryption (BIP38) CLI
- [lean11](https://github.com/carlosrabelo/lean11) -- Windows 11 debloater and ISO optimizer
